js/ui/handler/keyboard.js

Keyboard

The Keyboard handler responds to keyboard input by zooming, rotating, or panning the map. The following keyboard shortcuts are supported:

  • = / +: increase zoom level by 1
  • Shift-= / Shift-+: increase zoom level by 2
  • -: decrease zoom level by 1
  • Shift--: decrease zoom level by 2
  • Arrow keys: pan by 80 pixels
  • Shift+⇢: increase rotation by 2 degrees
  • Shift+⇠: decrease rotation by 2 degrees
  • Shift+⇡: increase pitch by 5 degrees
  • Shift+⇣: decrease pitch by 5 degrees

Examples

// Disable the keyboard handler
  map.keyboard.disable();

// Enable the keyboard handler
  map.keyboard.enable();
js/geo/lng_lat.js

LngLat (lng, lat)

Create a longitude, latitude object from a given longitude and latitude pair in degrees. Mapbox GL uses Longitude, Latitude coordinate order to match GeoJSON.

Note that any Mapbox GL method that accepts a LngLat object can also accept an Array and will perform an implicit conversion. The following lines are equivalent:

map.setCenter([-73.9749, 40.7736]);
map.setCenter( new mapboxgl.LngLat(-73.9749, 40.7736) );
parameter type description
lng number

longitude
lat number

latitude

Examples

var ll = new mapboxgl.LngLat(-73.9749, 40.7736);

js/ui/map.js

featuresIn ([bounds], params, callback, err, features)

Get all features in a rectangle.

Note: because features come from vector tiles, the returned features will be:

  1. Truncated at tile boundaries.
  2. Duplicated across tile boundaries.

For example, suppose there is a highway running through your rectangle in a featuresIn query. featuresIn will only give you the parts of the highway feature that lie within the map tiles covering your rectangle, even if the road actually extends into other tiles. Also, the portion of the highway within each map tile will come back as a separate feature.

parameter type description
bounds [Array<Point> or Arraynumber>>]

Coordinates of opposite corners of bounding rectangle, in pixel coordinates. Optional: use entire viewport if omitted.
params Object
params.layer string

Optional. Only return features from a given layer
params.type string

Optional. Either raster or vector
callback featuresAtCallback

function that receives the response
err Object or

Error If any
features Array

A JSON array of features given the passed parameters of featuresIn

Returns

Map :

this

js/source/video_source.js

VideoSource ([options])

Create a Video data source instance given an options object

parameter type description
options [Object]
options.urls Array<string>

An array of URLs to video files
options.coordinates Array

lng, lat coordinates in order clockwise starting at the top left: tl, tr, br, bl

Examples

var sourceObj = new mapboxgl.VideoSource({
   url: [
       'https://www.mapbox.com/videos/baltimore-smoke.mp4',
       'https://www.mapbox.com/videos/baltimore-smoke.webm'
   ],
   coordinates: [
       [-76.54335737228394, 39.18579907229748],
       [-76.52803659439087, 39.1838364847587],
       [-76.5295386314392, 39.17683392507606],
       [-76.54520273208618, 39.17876344106642]
   ]
});
map.addSource('some id', sourceObj); // add
map.removeSource('some id');  // remove
